# فروشگاه حیوانات خانگی - Pet Shop

یک اپلیکیشن Next.js فول‌استک با قابلیت وبلاگ به زبان فارسی

## ویژگی‌ها

- ✅ صفحه اصلی
- ✅ وبلاگ عمومی برای مشاهده پست‌ها
- ✅ پنل مدیریت برای ایجاد و ویرایش پست‌ها
- ✅ پشتیبانی کامل از زبان فارسی (RTL)
- ✅ رابط کاربری مدرن و زیبا

## نصب و راه‌اندازی

1. نصب وابستگی‌ها:
```bash
npm install
```

2. اجرای پروژه در حالت توسعه:
```bash
npm run dev
```

3. باز کردن مرورگر در آدرس:
```
http://localhost:3000
```

## ساختار پروژه

- `/app` - صفحات و روت‌های Next.js
- `/components` - کامپوننت‌های قابل استفاده مجدد
- `/lib` - توابع و منطق برنامه (مانند مدیریت پست‌ها)
- `/messages` - فایل‌های ترجمه

## نکات مهم

- در حال حاضر از یک دیتابیس درون‌حافظه استفاده می‌شود. برای استفاده در تولید، باید یک دیتابیس واقعی (مانند PostgreSQL یا MongoDB) اضافه کنید.
- پست‌ها در حافظه ذخیره می‌شوند و با ری‌استارت سرور پاک می‌شوند.

## اسکریپت‌ها

- `npm run dev` - اجرای سرور توسعه
- `npm run build` - ساخت نسخه تولید
- `npm run start` - اجرای نسخه تولید
- `npm run lint` - بررسی کد
